# AI Workflows - FOOD AI

Tài liệu mô tả các luồng xử lý dữ liệu và logic của Agent AI.

## 1. Luồng xử lý yêu cầu tìm kiếm (Search Workflow)
1. User Input -> Embedding -> Vector Search -> Top-K Results -> RAG Prompt -> Final Answer.

## 2. Luồng xử lý hội thoại (Conversation Workflow)
1. User Message -> Context Retrieval -> Intent Detection -> Action Selection -> AI Response.
